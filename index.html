GitHub Viewer Help

This application allows you to browse GitHub repositories in a classic Windows 3.11 interface.

Main Features:

  • Browse and search repositories
  • View repository details
  • Sort repositories by name, update date, stars, or size
  • Change view between large and small icons
  • Toggle between light and dark mode

Getting Started:

Double-click on a program group to see repositories in that category. Use the toolbar to search, sort, or change views.
