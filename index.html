Windows 3.11 GitHub Viewer Help
Getting Started

The GitHub Viewer allows you to browse repositories in a nostalgic Windows 3.11 interface. Repositories are organized into program groups for easy access.

Using Program Manager

Double-click on any repository icon to view its details. Right-click for additional options. Use the File menu to refresh the repository list or exit the application.

Keyboard Shortcuts

F5: Refresh repositories
Ctrl+F: Focus search box
Esc: Close dialogs
Arrow keys: Navigate repositories
Enter: Open selected repository
T: Toggle light/dark mode
